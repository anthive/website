# AntHive.IO
AntHIve.IO is a turn-based coding game. For detailed rules please visit [Rules Section ](https://anthive.io/rules/)

## Preview
https://anthive.netlify.app/

## Run locally
```
npm i
npm run dev
```

## .ENV
```
API_URL=
PROFILE_URL=
SANDBOX_STORAGE=
SIM_VERSION=5.0
GAMES_STORAGE=
WEBSITE_URL=
```
